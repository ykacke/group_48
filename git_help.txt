======>>> Первые три команды пишутся только один раз когда вы установили git на свой компьютер

git --version - показывает версию установленной программы гит
git config --global user.name "ВАШ ЛОГИН"
git config --global user.email "ВАШ EMAIL"
git config --list - показывает сохраненные настройки гита

======>>> 1 раз при создании нового проекта

git init - инициализирует проект(.git)
git remote add origin <HTTPS link.git> - соединяет репозиторий и ваш проект

======>>> команды, которые пишем каждый раз когда хотим отправить изменения проекта в репозиторий:

git add file_name или git add . - готовит ваши изменения к сохранению
git commit -m 'lesson 1' - окончательно сохраняет изменения
git push origin master - отправляет сохраненные изменения (коммиты) на сервер в репозиторий
